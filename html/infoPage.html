Anti-Phishing Extension

Created by Alexander D. Hughes

Open-Source

This program is open source! Meaning the original source code is freely available for anyone to read or modify, Go crazy!

No Tracking

Fishy Phish does not track any of your internet browsing and does not require any registration information.

Easy to use!

Incredibly easy to install either through Chrome web store or GitHub!

Setup Guide

Fishy Phish can be easily installed either through GitHub or the Google Chrome store!

Once installed, the extension will scan your emails for malicious links and grant you a smart popup if any are found. If the popup makes a mistake then you can easily add it to your personal safe list!

GitHub Installation

  • Download the GitHub repository.
  • Load up extensions on Google Chrome. (Top right corner)
  • Enable Developer Mode.
  • Load unpacked and look for the downloaded GitHub folder.
  • Make sure it is enabled and be safe!
