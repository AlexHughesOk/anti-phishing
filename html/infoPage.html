Fishy Phish

Anti-Phishing Extension

Created by Alexander D. Hughes

Open-Source

This program is open source! Meaning the original source code is freely available for anyone to read or modify, Go crazy!

No Tracking

Fishy Phish does not track any of your internet browsing and does not require any registration information.

Easy to use!

Incredibly easy to install either through Chrome web store or GitHub!

GitHub Installation
  1. Download the GitHub repository.
  2. Load up extensions on Google Chrome.
  3. Enable Developer Mode
  4. Load unpacked and look for the GitHub repository folder.
  5. Make sure it is enabled and be safe!
